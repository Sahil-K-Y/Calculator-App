## 🎯 Overview

A **feature-rich calculator** built with vanilla JavaScript. It uses a **safe math evaluation engine** (no `eval()`), supports **keyboard input**, and features a stunning **glassmorphism design** with smooth animations.

---

## ✨ Features

| Feature | Description |
|:---|:---|
| 🔢 **Basic Operations** | Addition, Subtraction, Multiplication, Division |
| 🔬 **Advanced Operations** | Percentage, Sign toggle, Decimal support |
| 🛡️ **Safe Math Engine** | Custom parser — no dangerous `eval()` |
| ⌨️ **Keyboard Support** | Full keyboard navigation & input |
| 💎 **Glassmorphism UI** | Frosted glass aesthetic with smooth animations |
| 📱 **Responsive** | Works on all screen sizes |
| ✅ **Error Handling** | Division by zero, invalid input protection |

## ⌨️ Keyboard Shortcuts

| Key | Action |
|:---:|:---|
| `0-9` | Number input |
| `+`, `-`, `*`, `/` | Operations |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last digit |
| `Escape` | Clear all |
| `.` | Decimal point |
